# Alcance del Proyecto

## Alcance Funcional

El proyecto contempla el desarrollo de una solución analítica para la predicción de churn en clientes corporativos de una empresa de telecomunicaciones.

La solución incluirá:

* Integración de fuentes de datos empresariales.
* Procesamiento y limpieza de información.
* Análisis exploratorio de datos.
* Construcción de variables predictivas.
* Entrenamiento y evaluación de modelos de Machine Learning.
* Generación de indicadores de riesgo de churn.
* Desarrollo de dashboard de visualización.

## Fuentes de Datos Consideradas

* empresas.csv
* servicios.csv
* consumo.csv
* facturacion.csv
* tickets.csv
* eventos_red.csv
* churn.csv

## Alcance Técnico

Tecnologías previstas:

* Python
* Pandas
* NumPy
* Scikit-Learn
* XGBoost
* Jupyter Notebook
* GitHub
* Power BI

## Exclusiones

El proyecto no contempla:

* Implementación en producción.
* Integración con sistemas transaccionales reales.
* Automatización de campañas comerciales.
* Procesamiento en tiempo real.
* Despliegue en nube.

## Resultado Esperado

Obtener un modelo predictivo capaz de estimar la probabilidad de abandono de clientes corporativos y proporcionar información útil para la ejecución de estrategias de retención.
