# Cronograma del Proyecto

| Semana | Actividad Principal                     | Entregable                               |
| ------ | --------------------------------------- | ---------------------------------------- |
| 1      | Definición del problema y planificación | Problema de negocio, objetivos y alcance |
| 2      | Análisis de fuentes de datos y modelado | Diccionario de datos y modelo relacional |
| 3      | ETL y calidad de datos                  | Dataset procesado                        |
| 4      | Análisis exploratorio de datos (EDA)    | Hallazgos y visualizaciones              |
| 5      | Feature Engineering                     | Dataset de variables predictivas         |
| 6      | Entrenamiento y evaluación de modelos   | Métricas y selección del modelo          |
| 7      | Desarrollo de dashboard                 | Dashboard Power BI                       |
| 8      | Informe final y sustentación            | Informe y presentación final             |

## Hitos del Proyecto

### Hito 1

Definición del problema de negocio y objetivos.

### Hito 2

Disponibilidad de datos limpios y consolidados.

### Hito 3

Construcción de variables predictivas.

### Hito 4

Entrenamiento del modelo de Machine Learning.

### Hito 5

Implementación del dashboard y presentación final.

## Fecha de Finalización Esperada

Al finalizar la semana 8 se contará con un modelo predictivo de churn para clientes B2B de telecomunicaciones y un dashboard para el análisis de riesgo de abandono.
